# Editorial Review Checklist

**Version:** 1.0  
**Status:** Draft  
**Last Updated:** July 2026  

---

This document is the companion to the Editorial Constitution (Article XI — Gold Standard Review). It provides concrete yes/no criteria and scoring guidelines for each review phase. The goal is to ensure consistent application of Gold Standard Review across all editors, reviewers, and chapters.

---

## How to Use This Checklist

1. Each phase has a set of criteria. Reviewers answer **Yes** or **No** for each.
2. A **No** answer must include a note explaining what is required for the item to pass.
3. Scoring thresholds define whether the phase passes overall.
4. All seven phases must pass before the chapter may be marked Published.
5. Results are recorded in the chapter's editorial metadata.

---

## Phase 1 — Expert Review

**Objective:** Subject-matter experts assess accuracy, fairness, and completeness.

**Required Participants:** Minimum 2 experts from different disciplinary or interpretive perspectives.

| # | Criterion | Yes/No | Notes |
|---|-----------|--------|-------|
| 1.1 | Are all factual claims in the chapter accurate according to the expert's knowledge? | | |
| 1.2 | Are the scholarly interpretations presented fairly (not as straw men)? | | |
| 1.3 | Are the scholars cited in the chapter correctly attributed with their actual positions? | | |
| 1.4 | Are there any significant omissions — evidence, arguments, or sources that should have been included? | | |
| 1.5 | Is the chapter free of errors that would embarrass the platform if publicly identified by this expert? | | |
| 1.6 | Does the expert agree that the chapter meets professional scholarly standards for accuracy? | | |
| 1.7 | Does the expert identify any claim that they believe is contradicted by the weight of evidence? | | |
| 1.8 | Are all areas of expert disagreement between the two (or more) expert reviewers documented? | | |

**Pass condition:** All criteria score Yes, or identified issues are documented and accepted by the Editor-in-Chief.

**Score:** ___ / 8

## Phase 2 — Reader Review

**Objective:** Four reader profiles identify confusion points, not satisfaction.

**Required Participants:** One each from: UPSC aspirant, MA History student, journalist, curious layperson.

**Method:** Ask only "Where did you become confused?" Do NOT ask "Did you like it?"

| # | Criterion | Yes/No | Notes |
|---|-----------|--------|-------|
| 2.1 | UPSC aspirant report reviewed. All confusion points documented. | | |
| 2.2 | MA History student report reviewed. All confusion points documented. | | |
| 2.3 | Journalist report reviewed. All confusion points documented. | | |
| 2.4 | Curious layperson report reviewed. All confusion points documented. | | |
| 2.5 | Confusion points analysed and categorised (terminology unclear? missing context? logical gap? assumption not justified?). | | |
| 2.6 | At least one revision made in response to reader confusion (or justification written for why no change is needed). | | |

**Pass condition:** All six criteria pass. Confusion points are resolved or explicitly noted as unresolved with explanation.

**Score:** ___ / 6

## Phase 3 — Evidence Audit

**Objective:** Every claim independently verified against cited sources. Counter-evidence checked.

**Reviewer:** Fact-checker who is not the writer.

| # | Criterion | Yes/No | Notes |
|---|-----------|--------|-------|
| 3.1 | Every claim in the chapter links to a Claim Registry entry with a valid Claim ID. | | |
| 3.2 | Every Claim Registry entry has at least one evidence reference from the Evidence Hierarchy. | | |
| 3.3 | Every evidence reference correctly supports the claim it is cited for (audited by reviewing the original source). | | |
| 3.4 | No evidence is cited out of context (audited by reviewing surrounding text in the source). | | |
| 3.5 | All Level 1 (primary document) citations are checked against the original document, not a secondary paraphrase. | | |
| 3.6 | Counterarguments are documented for every claim with a confidence score of "contested" or "weak." | | |
| 3.7 | Confidence scores are appropriate given the available evidence. | | |
| 3.8 | No claim contradicts its cited evidence. | | |
| 3.9 | All claims that the writer identified as "established" are genuinely supported by multiple independent Level 1–2 sources. | | |
| 3.10 | All sources in the chapter's source list are real, accessible, and correctly cited. | | |

**Pass condition:** All ten criteria score Yes. Any failure requires correction before proceeding.

**Score:** ___ / 10

## Phase 4 — Bias Audit

**Objective:** Systematic review for six categories of bias.

**Reviewer:** Editor who is not the writer.

| # | Bias Type | Criterion | Yes/No | Notes |
|---|-----------|-----------|--------|-------|
| 4.1 | Nationalist bias | Does the chapter present any nation's perspective as inherently more valid without supporting evidence? | | |
| 4.2 | Nationalist bias | Are the positions of all relevant parties presented with equal evidentiary scrutiny? | | |
| 4.3 | Imperial bias | Does the chapter avoid adopting the coloniser's perspective as the default or neutral position? | | |
| 4.4 | Imperial bias | Are colonial sources identified as products of their institutional context? | | |
| 4.5 | Presentism | Are historical actors judged by the standards and knowledge available at the time, not by contemporary standards? | | |
| 4.6 | Presentism | Does the chapter acknowledge when it is making a presentist judgment (e.g., "by today's standards, this was...")? | | |
| 4.7 | Hindsight bias | Does the chapter treat known outcomes as having been avoidable or inevitable, rather than one outcome among many that were possible at the time? | | |
| 4.8 | Hindsight bias | Are counterfactuals and contingencies presented alongside the narrative of what actually happened? | | |
| 4.9 | Selection bias | Is the evidence base drawn from multiple sources with different perspectives, not cherry-picked to support a single conclusion? | | |
| 4.10 | Selection bias | Are sources that disagree with the chapter's interpretive position cited and engaged with? | | |
| 4.11 | Confirmation bias | Would someone who disagrees with the chapter's interpretive conclusions recognise their position as having been fairly presented? | | |
| 4.12 | Confirmation bias | Has the reviewer actively searched for evidence that contradicts the chapter's claims? | | |

**Pass condition:** Minimum 10/12 Yes. The two No items must be documented with explanation and accepted by the Editor-in-Chief.

**Score:** ___ / 12

## Phase 5 — Visual Audit

**Objective:** Every visual reviewed for provenance, legality, pedagogical purpose, and accessibility.

| # | Criterion | Yes/No | Notes |
|---|-----------|--------|-------|
| 5.1 | Every visual has documented provenance (creator, date, location). | | |
| 5.2 | Every visual has a confirmed license or copyright status permitting use. | | |
| 5.3 | Every visual has an editorial caption explaining its historical significance (not merely descriptive). | | |
| 5.4 | Every visual has a "purpose" note explaining what it teaches. If it only decorates, it is removed. | | |
| 5.5 | Every visual has alt text conveying all information present in the image. | | |
| 5.6 | Every visual links to specific claims in the Claim Registry that it supports. | | |
| 5.7 | All charts and data visualisations include source data and methodology. | | |
| 5.8 | All maps include scale, projection, and legend. | | |
| 5.9 | No AI-generated images are used for historical reporting. | | |
| 5.10 | Any AI-generated diagrams are clearly labelled as such. | | |
| 5.11 | Visuals meet WCAG AA colour contrast requirements. | | |
| 5.12 | The visual acquisition list has been fully executed (i.e., acquisition list items are now actual curated assets, not placeholders). | | |

**Pass condition:** All criteria score Yes. No visual may be published without meeting all requirements.

**Score:** ___ / 12

## Phase 7 — Defensibility Audit

**Objective:** "Could we defend this?" For every major interpretive claim, verify the chain.

**Reviewer:** Editor-in-Chief.

### Audit Process

For each major interpretive claim in the chapter, confirm:

| # | Step | Yes/No | Notes |
|---|------|--------|-------|
| 7.1 | The claim is identified (list of major interpretive claims generated). | | |
| 7.2 | For each claim: the evidence is identified and verifiable. | | |
| 7.3 | For each claim: the primary source behind the evidence is identified. | | |
| 7.4 | For each claim: the scholarly disagreement around the claim is documented (who disagrees, on what grounds, with what evidence). | | |
| 7.5 | For each claim: the editorial reasoning for the chosen wording is documented (why this framing, not another). | | |
| 7.6 | For each claim: the editor can articulate why this claim would survive a public challenge. | | |
| 7.7 | No claim relies on a single, easily contested source. | | |
| 7.8 | The chapter's interpretive arc (if any) is supported by the cumulative weight of evidence, not by a single claim. | | |
| 7.9 | The chapter clearly distinguishes between established facts, scholarly interpretations, and editorial analysis. | | |
| 7.10 | The chapter explicitly acknowledges its limitations, open questions, and areas of scholarly disagreement. | | |

**Pass condition:** All ten criteria score Yes.

**Score:** ___ / 10

|--------|---------|
