# Confidence Register

**Global claim confidence tracking for The Breakdown Knowledge Platform.**

**Version:** 1.0  
**Status:** Active  
**Last Updated:** July 2026  

---

This register tracks confidence scores for every claim in the Claim Registry. Unlike chapter-level metadata (which reflects overall chapter confidence), this register provides claim-level global confidence that persists across all knowledge objects that reference the claim.

Confidence scores follow the Evidence Constitution (Article IV):

| Score | Meaning |
|-------|---------|
| Established (95–100) | Multiple independent Level 1–2 sources. No serious scholarly dispute. |
| Strong (80–94) | Strong evidence. Some disagreement but weight of evidence supports. |
| Moderate (60–79) | Evidence exists but is limited, contradictory, or subject to reasonable alternative interpretation. |
| Weak (40–59) | Limited evidence. Plausible but not firmly established. |
| Contested (0–39) | Significant scholarly disagreement. Evidence does not clearly favour one position. |
