# Knowledge Library — Editorial Style Guide

## 1. Voice and Tone

| Attribute | Rule |
|-----------|------|
| Voice | Authoritative but not academic. Write for an educated general reader. |
| Tone | Neutral, measured, evidence-first. Never polemical. |
| Register | Formal but accessible. Avoid jargon without explanation. |
| Person | Third person. Never first person ("we believe", "we think"). |
| Tense | Past tense for historical events. Present tense for enduring facts. |

## 2. Neutrality Rules

### 2.1 Distinguish Categories

Every statement in a chapter must be categorisable as one of:

| Category | Definition | Label |
|----------|------------|-------|
| **Fact** | Verifiable from a Tier 1–2 source | ✓ Fact |
| **Analysis** | Reasoned interpretation by the author | ◆ Analysis |
| **Perspective** | Viewpoint of a named person or school | ■ Perspective |
| **Speculation** | Projection or hypothetical (clearly marked) | ? Speculation |

### 2.2 Fairness

- All major perspectives on a contested issue must be represented
- Do not set up straw arguments. State the strongest version of each opposing view.
- When a policy is widely criticised globally, state that — even if the Indian government position differed.
- Give proportionate weight: fringe theories are not equivalent to mainstream scholarship.

### 2.3 Language Traps

| Avoid | Prefer |
|-------|--------|
| "India rightly decided..." | "India decided..." |
| "Nehru's wise statesmanship" | "Nehru argued..." |
| "The foolish opposition claimed" | "The opposition argued" |
| "Obviously" | Delete. If it's obvious, the evidence shows it. |
| "Regrettably" | Delete. State what happened and let the reader judge. |
| "Brave/chicken/weak/strong" | Describe actions. Do not add moral colour. |

## 3. Citation Format

### 3.1 Inline Citations

- Use numbered inline brackets: `[1]`
- Multiple citations: `[1, 3–5]`
- After the claim, not in the middle of a sentence when avoidable.

### 3.2 Full Citation Entry

```
[1] Title of Source, Author Name, Publication/Journal,
    Publisher, Date, Page(s). URL or Archive Link. Accessed: 2026-01-15.
```

### 3.3 Primary Source Flag

Primary sources (Tier 1) are prefixed with a document icon in the source list:

```
📄 Treaty of Peace, Friendship and Cooperation, USSR–India, August 9, 1971.
```

### 3.4 Translations

- Use standard English translations for non-English titles
- Provide original title in parentheses on first citation
- For Indian languages, provide transliteration

### 3.5 Archival

- All URLs must have an archive.org snapshot or equivalent
- Declassified documents include declassification date
- Government records include file number where available

## 4. Evidence Labels

Each evidence reference in a claim block includes a relevance label:

| Label | Meaning |
|-------|---------|
| Direct | Explicitly supports the claim |
| Supporting | Provides contextual support |
| Contextual | Background information |

## 5. Fact vs Analysis Markers

In the rendered chapter, visual markers distinguish content types:

```
[F] This is a verifiable fact cited to a primary source.
[A] This is the author's interpretation of the facts.
[P: Name] This is the perspective of the named thinker or school.
[S] This is a projection or hypothetical scenario.
```

At Explorer depth, these markers are hidden. At Researcher depth, every block is labelled.

## 6. Common Editorial Pitfalls

| Pitfall | Fix |
|---------|-----|
| Presenting opinion as fact | Add "[A]" marker or source citation |
| False balance (30% view vs 70% view as equal) | Reflect actual weight of scholarly consensus |
| Naming only one side | Add counter-perspective or explicitly acknowledge absence |
| Retrospective determinism | State the alternatives that existed at the time |
| Hagiography | Critique honestly. Even founders made mistakes. |
| Presentism | Evaluate in historical context, not modern values alone |
| Source over-reliance on one type | Balance official, academic, and journalistic sources |

## 7. Names and Terminology

### 7.1 Official Names

- Use official English names for treaties, organisations, and institutions
- First mention: full name, then acceptable abbreviation: "Non-Aligned Movement (NAM)"
- Maintain a glossary per collection

### 7.2 Sensitive Terms

| Term | Policy |
|------|--------|
| Pakistan | "Pakistan", not "PoK" or "West Pakistan" (historical contexts excepted) |
| Kashmir | "Jammu and Kashmir" for the pre-2019 state; precise terms for current status |
| China | "People's Republic of China" (PRC) for official contexts |
| Taiwan | "Taiwan" with note on status where relevant |
| East Pakistan | "East Pakistan" (pre-1971); "Bangladesh" (post-1971) |
| Burma → Myanmar | Use historical name for the period referenced |
| Soviet Union | "Soviet Union" or "USSR" consistently |

### 7.3 Neutral Descriptors

- "Insurgency" / "militant group" / "armed group" — use third-party UN or academic classification
- Avoid "terrorist" unless quoting a named official source or legal designation
- When quoting a designation, attribute it: "the US State Department designated..."

## 8. Dates

- Format: "21 January 1956"
- Decades: "the 1960s" (not "the 60s" or "1960's")
- Centuries: "the twentieth century" (not "the 1900s")
- Eras: "1947–1962" with en-dash

## 9. Controversial Content

- When covering disputed events, present each side's version as perspective, not fact
- Clearly state what is agreed upon by all parties and where disagreements begin
- If official Indian records and foreign records disagree, state both with sources

## 10. Research Methodology Content

Every collection must publish a Research Methodology page before any chapter. It documents:

### 10.1 Transparency
- What sources were consulted and why
- What sources were excluded and why
- How conflicting evidence is adjudicated
- What confidence levels mean
- What remains unknown

### 10.2 The "What Cannot Be Known" Section
This is critical for trust. Explicitly acknowledge:
- Records still classified by the Ministry of External Affairs
- Periods where Indian archival access is restricted
- Gaps in non-English source translation
- Events where no reliable primary source exists

### 10.3 Disclosure
- Author credentials and potential biases
- Funding sources for the collection
- Editorial review process
- Peer reviewers (if any)

## 11. Historiography Guidance

### 11.1 When to Include
Every major event (war, treaty, crisis, election, summit) should have a historiography section explaining how interpretations have changed.

### 11.2 Required Perspectives
For India and the World, each historiography must include (where available):
- Official Indian government position (MEA histories, official statements)
- Counterpart country position (Chinese, Pakistani, Soviet archives)
- Western academic scholarship
- Indian academic scholarship (including non-English)
- Postcolonial / subaltern critiques
- Recent revisionist scholarship

### 11.3 How to Write It
- Present each interpretation in neutral language
- Identify the key scholars or institutions associated with each view
- Note when and why interpretations shifted (e.g., post-Cold War archival openings)
- Do not adjudicate — let the reader see the landscape of debate

## 12. Counterfactual Writing Rules

### 12.1 Separation
Counterfactuals must be visually and structurally separated from the evidence-based narrative. A labelled box or distinct section with a clear header: "Counterfactual Analysis — Not Historical Record."

### 12.2 Sourcing
Even counterfactuals require sources. Reference scholarly works that engage in counterfactual methodology (e.g., Richard Ned Lebow, Philip Tetlock, Niall Ferguson for methodology).

### 12.3 Language
| Do | Don't |
|----|-------|
| "If India had joined NATO..." | "India should have joined NATO" |
| "Counterfactual analysis suggests..." | "India would have been safer if..." |
| "Scholars who examine this question argue..." | "Obviously, India made a mistake..." |

### 12.4 Placement
Counterfactuals appear at the end of a chapter or in a dedicated appendix volume — never interleaved with the factual narrative.

## 13. Confidence Scoring Framework

### 13.1 How Confidence Is Determined

| Score | Meaning | Criteria |
|-------|---------|----------|
| 95–100% | Established | Multiple Tier 1 sources, no contradictions |
| 85–94% | Well-supported | Tier 1–2 sources, minor disagreements |
| 70–84% | Probable | Tier 2–3 sources, some debate |
| 50–69% | Contested | Significant scholarly disagreement |
| <50% | Speculative | Limited sources, primarily interpretation |

### 13.2 Display Rules
- Overall chapter confidence is an aggregate of all claim confidences
- Individual claims can have lower confidence than the chapter aggregate
- Contradictory sources reduce confidence proportionally
- Confidence is recalculated at each freshness check

## 14. Review Checklist

Before any chapter is published:

- [ ] Every substantive claim has at least one citation
- [ ] Sources have been accessed and confirmed
- [ ] Rival perspectives are represented
- [ ] No editorialising ("rightly", "unfortunately", "obviously")
- [ ] All URLs have archive.org snapshots
- [ ] Media licenses are confirmed
- [ ] Glossary terms are defined
- [ ] The chapter has been read by a second editor
- [ ] Historiography section written for major events
- [ ] Decision matrix completed for major policy decisions
- [ ] Confidence score calculated and documented
- [ ] Reading difficulty rated with prerequisites listed
- [ ] Counterfactual sections (if any) clearly separated and labelled
- [ ] Research Methodology page is published and linked
- [ ] "How to Read This" page is published and linked
