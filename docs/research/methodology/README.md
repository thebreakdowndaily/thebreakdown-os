# Research Methodology v1.0

Welcome to the **Research Methodology v1.0**. This directory contains the definitive specification for how the Breakdown platform conducts research, evaluates evidence, attributes responsibility, and manages lifecycle states.

- **Version**: 1.0
- **Status**: Draft (awaiting editorial sign‑off)
- **Effective date**: 2026‑07‑20
- **Owner**: The Breakdown Research & Editorial
- **Scope**: All research, verification, and knowledge‑object creation processes across the platform.

## Contents

- `v1.0.md` – Full methodology specification.
- `adversarial-tests.md` – Structured adversarial test scenarios used to validate the methodology.

Use this directory as the single source of truth for any research‑related rules or vocabularies.
