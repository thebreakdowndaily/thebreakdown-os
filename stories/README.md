**Goal:** Eight stories, eight weeks. Each story answers one reader question and leads naturally to the next. A reader who finishes Week 1 should want Week 2.

## Definition of Done

A story is done only when it reaches:

```
Idea          ✓
Research      ✓
Verification  ✓
Knowledge     ✓
Architecture  ✓
Narrative     ✓
Review        ✓
Learning      ✓
Publication   ✓
Observation   ✓
First Correction Cycle  ✓
```

## Freeze rule

No structural changes to the Editorial OS during Season 1 unless a workflow blocker prevents publication. Minor bug fixes are fine. Architectural improvements wait until after the season. Keep operational observations in each story's `journal.md`.
