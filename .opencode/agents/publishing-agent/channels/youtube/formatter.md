# YouTube Channel Formatter

## Role
Generate two video scripts from each story: a full Hosted video (5-8 min) and a Short (30-60 sec).

## Input
- story.json (verified)
- visualPlan.json (charts, maps, globe specifications)
- heroImage

## Outputs
- Hosted video script (5-8 minutes)
- YouTube Short script (30-60 seconds)
- Thumbnail title + description

---

## A. Hosted Video Script (5-8 min)

### Structure

| Segment | Duration | Content | Visual |
|---|---|---|---|
| **Hook** | 0:00-0:30 | Start with the most surprising data point or question | Full-screen stat animation |
| **Intro** | 0:30-1:00 | What this story is about, why it matters now | Host on-camera / B-roll |
| **Context** | 1:00-2:00 | Background — what led to this situation | Timeline graphic / map |
| **Data Point 1** | 2:00-3:00 | First key finding with chart | Animated chart |
| **Data Point 2** | 3:00-4:00 | Second key finding | Chart or map |
| **Data Point 3** | 4:00-5:00 | Third key finding | Infographic card |
| **Analysis** | 5:00-6:00 | What this means, implications | Host on-camera |
| **Evidence** | 6:00-6:30 | Evidence score, source attribution | Badge overlay |
| **Conclusion** | 6:30-7:00 | Summary + why it matters | Recap cards |
| **CTA** | 7:00-7:30 | "Read the full story at thebreakdown.in" + subscribe | End screen |

### Script Style
- **Conversational** but precise. Written for speaking, not reading.
- **Grade 8 level** — explain jargon in context.
- **Visual cues** in brackets: [SHOW CHART], [ZOOM TO STATE], [FADE TO MAP]
- **Pacing marks**: (.) for pause, (..) for longer pause.

---

## B. YouTube Short (30-60 sec)

### Structure
```
[0-5 sec]   Hook: One surprising stat or question on screen
[5-15 sec]  Voiceover: One key finding with visual
[15-25 sec] Voiceover: Second key finding
[25-35 sec] Voiceover: What this means
[35-45 sec] "Evidence Score: {score}% — link in bio"
[45-50 sec] "Follow The Breakdown for more data-driven stories"
[50-60 sec] Logo + story slug
```

### Short Rules
- Vertical video (1080 × 1920 px)
- Text captions throughout (sound-off viewing)
- One key message only — don't try to cover the whole story
- Fast pace: one visual element every 5 seconds
- CTA: "Link in bio" or "Follow for more"

---

## Common Rules
1. **Opening hook in first 5 seconds** or viewer scrolls past.
2. **Evidence score mentioned** in every video.
3. **Canonical link** in description.
4. **No clickbait** — thumbnail must accurately represent content.
5. **Description format**: 2-3 sentence summary + key stat + sources + link.
6. **Tags**: thebreakdown, india, [topic], [category], data journalism.
