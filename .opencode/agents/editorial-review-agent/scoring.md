# THE BREAKDOWN
## Editorial Review Agent — Scoring Model v1.0

### Overall Score

The overall score (0–100) is a weighted composite of the nine editorial score dimensions:

| Dimension | Weight |
|-----------|--------|
| Accuracy | 20% |
| Evidence | 15% |
| Research | 12% |
| Trust | 12% |
| Readability | 10% |
| Context | 10% |
| Structure | 8% |
| Originality | 8% |
| VisualPotential | 5% |

```
Overall = (Accuracy × 0.20) + (Evidence × 0.15) + (Research × 0.12) +
          (Trust × 0.12) + (Readability × 0.10) + (Context × 0.10) +
          (Structure × 0.08) + (Originality × 0.08) + (VisualPotential × 0.05)
```

---

### Quality Gates

Six mandatory gates. Each dimension is scored 0–100.

| Gate | Minimum | Scoring Guide |
|------|---------|---------------|
| **Research** | ≥ 90 | 95+ = Multiple Tier 1 sources, diverse perspectives. 90–94 = Adequate sources, some gaps. < 90 = Missing key sources, over-reliance on Tier 5. |
| **Evidence** | ≥ 90 | 95+ = Every claim cited, inline citations, source quality noted. 90–94 = Most claims cited, minor gaps. < 90 = Multiple unsupported claims. |
| **Accuracy** | ≥ 95 | 98–100 = No errors found. 95–97 = Minor non-factual errors (typo in name, formatting). < 95 = Any factual error. |
| **Readability** | ≥ 90 | 95+ = Grade 8–9, short paragraphs, clear transitions. 90–94 = Grade 10, occasional dense paragraph. < 90 = Above grade 10, wall-of-text paragraphs. |
| **Trust** | ≥ 95 | 97–100 = All sources Tier 1–2, verification passed, no bias. 95–96 = Minor trust concerns. < 95 = Low-quality sources, unverified claims, bias detected. |
| **Context** | ≥ 90 | 95+ = Historical context, causal explanations, stakeholder mapping. 90–94 = Basic context present. < 90 = Reader must have prior knowledge. |

**If any gate score is below the minimum, the story is rejected.** Not revision. Rejection.

### Editorial Score Dimensions — Detailed Rubric

#### Research (0–100)
| Range | Description |
|-------|-------------|
| 95–100 | Multiple Tier 1 sources, government data, expert interviews, diverse perspectives across political spectrum |
| 85–94 | Tier 1–2 sources, adequate coverage, one or two perspectives lightly covered |
| 70–84 | Tier 2–3 sources, limited perspectives, some reliance on news articles |
| 50–69 | Tier 3–4 sources, narrow perspective, heavy reliance on single source |
| < 50 | Tier 5 sources only, opinion-based, no primary sources |

#### Evidence (0–100)
| Range | Description |
|-------|-------------|
| 95–100 | Every claim immediately followed by citation. Inline evidence format. Source quality noted. |
| 85–94 | Most claims cited. Occasional delay between claim and source. |
| 70–84 | Evidence present but not always adjacent to claims. Some claims uncited. |
| 50–69 | Several uncited claims. Evidence often appears paragraphs later. |
| < 50 | Claims without evidence. Citations missing or generic ("studies show"). |

#### Accuracy (0–100)
| Range | Description |
|-------|-------------|
| 98–100 | No factual errors. Names, dates, numbers, quotes all verified. |
| 95–97 | Minor non-factual errors only (formatting, typo in non-critical name) |
| 90–94 | One factual error in a non-critical detail |
| 80–89 | One factual error in a critical detail, or multiple minor errors |
| < 80 | Multiple factual errors. Story cannot be trusted. |

#### Structure (0–100)
| Range | Description |
|-------|-------------|
| 95–100 | All architecture sections present, correct order, all questions answered, perfect narrative flow |
| 85–94 | All sections present, minor order deviation or one weak transition |
| 70–84 | One section missing or one question unanswered |
| 50–69 | Multiple sections missing or reordered |
| < 50 | Structure does not match architecture |

#### Readability (0–100)
| Range | Description |
|-------|-------------|
| 95–100 | Grade 8–9. Short paragraphs (2–3 sentences). Varied sentence length. Clear transitions. |
| 85–94 | Grade 9–10. Occasional long paragraph or unclear transition. |
| 70–84 | Grade 10–11. Several dense paragraphs. Transitions weak. |
| 50–69 | Grade 12+. Long paragraphs. Few transitions. Reader must re-read. |
| < 50 | Wall of text. Paragraphs > 100 words. Sentences > 25 words. No transitions. |

#### Context (0–100)
| Range | Description |
|-------|-------------|
| 95–100 | Full background: historical context, causal chain, stakeholder positions, why this matters now |
| 85–94 | Good context: background covered, causal links mostly clear |
| 70–84 | Basic context: some background, some unexplained gaps |
| 50–69 | Minimal context: reader needs prior knowledge |
| < 50 | No context: events presented without background or framing |

#### VisualPotential (0–100)
| Range | Description |
|-------|-------------|
| 95–100 | Architecture visual plan fully specified, every visual has a clear purpose |
| 85–94 | Visual plan present, some visuals lack clear purpose |
| 70–84 | Visuals mentioned but not specified |
| 50–69 | Few visual opportunities identified |
| < 50 | No visual plan |

#### Originality (0–100)
| Range | Description |
|-------|-------------|
| 95–100 | Fresh angle, new data, unique framing not seen in other coverage |
| 85–94 | Strong angle, mostly original framing |
| 70–84 | Standard angle, adequately covered |
| 50–69 | Recycled angle, no new information |
| < 50 | Identical to existing coverage |

#### Trust (0–100)
| Range | Description |
|-------|-------------|
| 97–100 | All sources Tier 1–2. Verification passed. No bias. Self-critique: meets Reuters standards. |
| 95–96 | Minor trust concern (one Tier 3 source, or very minor bias detected). |
| 85–94 | Multiple Tier 3 sources. Some verification gaps. |
| 70–84 | Reliance on Tier 4 sources. Bias concerns. |
| < 70 | Tier 5 sources. Verification failed. Bias detected. Cannot trust. |

---

### Scoring Quick Reference

**Gate formula:** Score 0–100 per dimension. Apply minimum threshold. Any fail = reject.

**Overall formula:** Weighted average of all nine dimensions.

**Pass threshold for revision:** All gates pass, overall ≥ 80, checklist items 1–6 all true.

**Pass threshold for approval:** All gates pass, overall ≥ 85, all checklist items true, no critical issues, self-critique passes.
