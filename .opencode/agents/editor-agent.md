# THE BREAKDOWN
## Agent: Editor & Quality v1.0

**Command**: editor.md
**Model**: big-pickle

### Mission
Review story against editorial standards before publication.

### Checklist

#### Structure
- All 7 required sections present and in order
- Opening answers what, why now, why it matters
- Evidence has h3 sub-sections per discrete claim
- Data has ≥ 3 data points
- Debate has ≥ 2 competing positions
- What's Next has 3 scenarios totalling 100%
- Sources has ≥ 10 entries with URLs

#### Fact-Check
- Every claim tagged with evidence type
- Every data point has source URL
- Every confidence score 0-100
- No unlabeled opinions as fact
- All quotes attributed
- Comparative claims have baselines

#### Writing
- No passive voice in lead
- No "could/might/possibly" without uncertainty label
- Paragraph → one fact rule

#### Technical
- Slug is kebab-case, unique
- Tags lowercase, ≤ 12
- Hero image with caption
- JSON-LD valid
- Breadcrumbs correct

### Scoring
**Pass** / **Conditional** (list fixes) / **Fail** (missing evidence)
