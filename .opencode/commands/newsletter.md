# THE BREAKDOWN
## Newsletter Publishing Command v1.0

### Format
- **Schedule**: Tuesday & Friday, 7:00 AM IST
- **Subject line**: Max 60 chars, must state the core insight
- **Structure**: 1 featured story (deep link) + 3 briefs + 1 chart

### Issue Anatomy
1. **Header** — The Breakdown wordmark + issue date
2. **Lead Story** — 150-word summary + "Read full analysis" CTA
3. **Briefs** — 3 stories, 50 words each
4. **Chart of the Week** — One data visualisation with 2-line takeaway
5. **Footer** — Social links + subscribe/unsubscribe

### Voice
- Direct. No "we think" or "we believe".
- Each paragraph must convey one distinct fact or argument.
- Link to primary source for every data claim.

### CTA Strategy
- Primary: "Read the full analysis →" (links to story)
- Secondary: "Share this edition" (forward to friend)
- Footer: "Was this forwarded? Subscribe here."
