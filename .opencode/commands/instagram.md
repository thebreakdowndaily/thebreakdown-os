# THE BREAKDOWN
## Instagram Publishing Command v1.0

### Brand Identity
- Handle: `@thebreakdowndaily`
- Primary: `#D4A843` (gold), Background: `#0A0A0F` (near-black), Text: `#F5F5F5`
- Headings: Bebas Neue, Body: Source Sans 3
- Tone: Authoritative but accessible. Evidence-first. No jargon.

### Post Types
1. **Story Promotion** — 4:5 image or 6-10 slide carousel
2. **Data Carousel** — 4-6 slides, one data point per slide
3. **Weekly Roundup** — Saturday, 5 stories in 5 slides
4. **Explainer Thread** — 8-10 slide deep dive

### Carousel Specs
- Aspect ratio: 4:5 (1080×1350 px)
- Cover: Hook + gold accent + story title
- Slides: Max 6 bullet points per slide
- Final slide: Save-and-share CTA

### Caption Structure
- Lead hook (1 line, bold claim or question)
- Context (2-3 lines)
- Story link in bio mention
- 8-12 hashtags (primary + category)

### Publishing Workflow
1. Create slides → 2. Write caption → 3. First comment with link
4. Schedule for 7PM IST → 5. Monitor first 2 hours for engagement
