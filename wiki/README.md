## How to Contribute

1. Find the right section for your content.
2. Create a new `.md` file or extend an existing one.
3. Link back from the relevant `README.md` index table.
4. Add a YAML frontmatter block:

```yaml
---
title: Page Title
author: Your Name
last_updated: YYYY-MM-DD
---
```
