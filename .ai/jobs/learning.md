# Job: learning

Produces the learning infrastructure for an approved chapter.

## Agent

`agents/learning-designer.md`

## Input

`artifacts/draft.md` + `artifacts/review.md` (from narrative and editorial jobs).

## Artifact

`artifacts/learning.md`

Contains: learning objectives (specific, verifiable), key questions restated with answers, glossary, further reading (curated), reflection prompts, continue-learning recommendations.

## Quality gate

Objectives are specific and verifiable. Glossary terms defined in reader-accessible language. Further reading curated (5–7 entries). Reflection prompts test understanding, not recall. Continue-learning links resolve to existing objects.

## State

`pending` → `running` → `complete` | `failed`
